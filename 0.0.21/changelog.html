Changelog¶

All notable changes to the Feather SDK should be recorded here.

How to use¶

  • During development: add entries under Unreleased.

  • When releasing X.Y.Z:

    • Move the relevant items from Unreleased into a new ## [X.Y.Z] section.

    • Keep sections grouped by Added / Changed / Deprecated / Removed / Fixed / Security (Keep a Changelog style).

    • After publishing, start fresh under Unreleased for the next cycle.

Format¶

This file follows Keep a Changelog.

[Unreleased]¶

[0.0.21]¶

Added¶

  • Support for both pico and adc board voltage dividers

  • Update configs: make sure the power/ config has the lines adc_type: "pico"  # "pico" or "adc" and hz: 8

Changed¶

  • robot.neck.move() changed to robot.neck.go_to() for consistency

[0.0.17]¶

Added¶

  • Docs release (https://docs.feather.dev/)

  • API Reference for base, torso, neck, and arms
